# Nueva Libreria (Adaptador) Freshchat

El objetivo de este proyecto es generar una libreria mas amigable para freshChat que tiene una libreria muy complicada de usar, la idea es que esta nueva libreria esta basada en caso de usos mas cotidianos con metodos como login, logout, changeUser en vez de los metodos complejos y viejos con los que actualmente cuenta.

De esta manera creando una nueva Libreria tambien deberia ser mas facil implementar en otros proyectos o frameworks.

Directorios

- freschat-app (directorio donde se desarrollara la nueva API)
- template (directorio de ejemplo con un caso de uso similar obtenido desde [aqui](https://community.freshworks.dev/t/integration-of-freshchat-with-react-web/11153/12)
